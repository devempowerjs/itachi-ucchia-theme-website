# Portfolio - Itachi Uchiha Theme

This is a personal portfolio website styled with a premium, anime-inspired dark theme based on Itachi Uchiha from Naruto. It features a modern layout with glowing effects, custom animations, and a sleek user interface.

##  Technologies Used

- **HTML5:** For the core structure and semantic markup.
- **CSS3 (Vanilla):** For all custom styling, including the dark theme (`itachi.css`), glassmorphism, glowing hover effects, CSS animations, and custom SVG cursors.
- **JavaScript (Vanilla):** Used for DOM manipulation, dynamic scrolling features, and a custom HTML5 Canvas particle system (Embers effect).
- **Bootstrap 5:** Used for the responsive grid layout, spacing utilities, and core component structure.
- **Boxicons & Bootstrap Icons:** Used for scalable vector icons throughout the project.
- **AOS (Animate On Scroll):** Used to trigger smooth fade and zoom animations as elements scroll into view.
- **Typed.js:** Used for the typing animation effect in the hero section.
- **HTML5 Canvas API:** Used to render the dynamic "floating embers" background particle effect on the landing page.

##  Features

- **Dark Mode Aesthetic:** Deep blacks and crimson reds inspired by the Akatsuki.
- **Glassmorphism UI:** Semi-transparent cards with background blur and glowing red borders for a premium feel.
- **Dynamic Embers Background:** An interactive canvas animation in the hero section.
- **Sharingan Preloader:** A custom spinning Sharingan eye replaces the standard loading spinner.
- **Custom Kunai Cursor:** The default mouse cursor is replaced with an SVG kunai.
- **Responsive Layout:** fully optimized for Desktop, Tablet, and Mobile devices using Bootstrap's grid.

##  Project Structure

- `index.html`: The main entry point of the website.
- `assets/css/style.css`: The original base template styles.
- `assets/css/itachi.css`: The overriding dark theme styles, animations, and custom aesthetics.
- `assets/js/main.js`: Contains all the JavaScript logic (AOS initialization, Canvas Embers, Waypoints).
- `assets/img/`: Contains all images, including the generated Itachi background and Sharingan assets.
- `assets/resume/`: Contains the downloadable PDF resume.
- `assets/vendor/`: Contains third-party libraries (Bootstrap, AOS, Boxicons, etc.).

##  How to Run

1. Clone or download this repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. No build tools or backend server are required—everything runs purely on the frontend.
